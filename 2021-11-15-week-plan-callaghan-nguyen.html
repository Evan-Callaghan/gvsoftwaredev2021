Class Assignment: Running a Class Session - Evan Callaghan & Jackson Nguyen

Our task for the class: Include a blurb description for each assignment

Monday:
  • First, we will designate a previous class assignment to each student group.
  • The pair will then spend the class period discussing/writing a four to five sentence description of the assignment and writing a few sentences about how they interpreted/attacked the problem at hand.
  • At the end of the class period, every student group will share what they wrote for their designated assignment.
Wednesday:
  • Students will spend the first five to ten minutes discussing their assignment descriptions and making the appropriate edits.
  • Then, the assignment descriptions from each group will be randomly distributed to another group for review, edits, and suggestions.
  • Again, we will randomly exchange assignment descriptions for review, edits, and suggestions.
  • After two other groups have made their edits, the descriptions will be returned to the original groups.
Friday:
  • To start the final day, groups will have five to ten minutes to discuss the suggestions and edits from other groups.
  • Then, each group will present their final description to the class.
  • Once everyone is happy with the assignment description, it can be added to the website as an abstract for each blog post so the reader is aware what topics are covered in the blog.
