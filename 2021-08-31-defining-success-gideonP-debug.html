Please follow instructions below to run this program in you PC:

  • Open this link in a new tab on your browser
  • Download all files from my GitHub folder to your local machine as a zip folder by clicking the green botton labeled "code".
  • Use winZip or WinRar application to extract the contents of the zip file to your local folder.
  • Open Students Search.html by double clocking on it.
  • Select student's name on the list then click Search button.
  • You will be able to see students information below as you select each student's name from the list.
